# 场次复盘验收

## 前置条件

- 选择一场同时具有回放、分钟指标、评论和 ASR 话术的真实场次。
- 场次详情接口与复盘工作台接口返回成功。

## 操作步骤

1. 从直播场次列表进入详情，确认首要区域是竖屏回放和统一复盘分析。
2. 在统一时间轴分别筛选实时告警、复盘发现、用户评论、主播话术和指标采样。
3. 点击有时间点的证据，确认播放器跳转到对应位置。
4. 在同一区域切换“分钟曲线”和“AI 分析”，确认不再出现第二份重复分钟趋势。
5. 更新一条复盘发现为已确认或已解决，刷新后核对状态持久化。
6. 选择同主播上一场进行跨场对比，核对真实场次 ID 和曲线。

## 预期结果

- 所有结论可追溯到真实指标、评论或话术证据；缺失数据明确显示，不猜测时间点。
- 详情页只保留一个分钟曲线入口，AI 分析与时间轴共享当前场次。
- 评论和观众画像作为补充数据保留，不阻断主复盘流程。
- 复盘发现可直接维护状态，历史整改任务数据不被删除。

## 失败与恢复

- 回放无画面：检查 H.265 转 H.264 服务与本地分片缓存日志。
- AI 返回无效结构：保留 Trace ID，不覆盖上一份可用报告。
- 证据时间不存在：标记该结论为待复核，禁止作为稳定结论使用。
