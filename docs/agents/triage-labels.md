# 分类标签

工程技能使用 5 个标准分类角色。本文件将这些角色映射到本仓库实际使用的标签名。

| 技能中的标签名         | 本仓库标签名        | 含义                         |
| ---------------------- | ------------------- | ---------------------------- |
| `needs-triage`         | `needs-triage`      | 维护者需要对这个问题进行评估 |
| `needs-info`           | `needs-info`        | 等待提问者补充更多信息       |
| `ready-for-agent`      | `ready-for-agent`   | 规格完整，可由 AI 代理处理   |
| `ready-for-human`      | `ready-for-human`   | 需要人工实现                 |
| `wontfix`              | `wontfix`           | 不会处理                     |

当技能提到某个角色时（例如"打上 ready-for-agent 标签"），使用本表右列对应的标签名。
